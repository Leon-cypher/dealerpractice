# DealerPro - 專業荷官訓練工具

這是一個專為德州撲克荷官與玩家設計的練習工具，旨在訓練快速核算底池、判斷牌力勝負以及理論知識理解。

## 核心功能

### 1. 挑戰模式 (Challenge Mode) 🔥
- **限時衝刺**：5 分鐘倒數計時，挑戰在極限時間下的準確度與手速。
- **公平競賽**：挑戰期間模式與種類將被鎖定，確保排行榜的公平基準。
- **累積計分**：挑戰模式下答錯僅重置「連勝加成」，不歸零總分，鼓勵在限時內爭取最高產出。

### 2. 計分與排行榜系統 🏆
- **五大獨立排行榜**：各分類獨立計算前 10 名門檻，互不干擾。
  - 底池計算
  - 德州判斷
  - 奧馬哈判斷
  - BIGO 判斷
  - 理論測驗
- **連勝加成 (Streak)**：每連續答對一題，下一題得分增加 **20%**。
- **速度獎勵 (Time Bonus)**：10 秒內完成答題享有 **1.5 倍** 加成。

### 3. 理論測驗系統 (Knowledge Quiz) 📚
- **110 題大全**：整合自專業題庫，包含規則判定、發牌程序與特殊情境處理。
- **即時解析**：答題後立刻顯示正確答案與詳細規則說明，強化學習記憶。

### 4. 底池分配練習 (Split Pot)
- **多邊池模擬**：模擬翻牌前 (Pre-flop) 多人 All-in 的複雜邊池計算。
- **餘數處理 (Odd Chips)**：遵循專業規則，將無法整除的零錢優先分配給 ID 較小的贏家。

### 5. 勝負判斷練習 (Showdown)
- **支援多種遊戲**：Texas Hold'em, Omaha (2+3), BIGO (5-Card Hi-Lo)。
- **動態教學**：即時顯示各遊戲種類的特殊規則（如 8-or-better 低牌規則）。

## 技術特點
- **TypeScript 驅動**：全系統強型別定義，確保邏輯嚴謹與程式碼穩定。
- **Supabase 整合**：雲端儲存與篩選分頁排行榜資料。
- **Tailwind CSS**：打造現代化、響應式且具備教學感的 UI 介面。
