# DealerPro - 專業荷官訓練工具

這是一個專為德州撲克荷官與玩家設計的練習工具，旨在訓練快速核算底池與判斷牌力勝負的能力。

## 核心功能

### 1. 挑戰模式 (Challenge Mode) - NEW! 🔥
- **限時衝刺**：5 分鐘倒數計時，挑戰在時間壓力下的準確度。
- **累積計分**：挑戰模式下答錯不歸零總分，僅中斷連勝加成。
- **動態計時器**：介面顯示顯眼的倒數時鐘，營造競技感。

### 2. 專業計分系統
- **連勝加成 (Streak Multiplier)**：每連續答對一題，下一題得分增加 **20%**，連續答對是拿高分的關鍵。
- **速度獎勵 (Time Bonus)**：10 秒內答對享有 **1.5 倍** 分數加成。
- **難度係數**：根據遊戲種類（德州 < 奧馬哈 < BIGO）提供不同的基礎分數倍率。

### 3. 底池分配練習 (Split Pot)
- **多邊池模擬**：自動產生多位玩家 All-in 的複雜情境，包含主池 (Main Pot) 與多個邊池 (Side Pot)。
- **餘數處理 (Odd Chips)**：遵循專業規則，將無法整除的零錢分配給按鈕位順時針第一位贏家。
- **核算回饋**：即時檢查底池金額與玩家獲配總額是否正確。

### 2. 勝負判斷練習 (Showdown)
- **多種遊戲支援**：
  - **Texas Hold'em**：傳統德州撲克規則。
  - **Omaha**：強制 2+3 規則（2 張手牌搭配 3 張公牌）。
  - **BIGO (5-Card Hi-Lo)**：五張手牌的奧馬哈高低牌規則，包含 8-or-better 低牌判定。
- **精確判定系統**：採用 15 進位權重算法，完美處理所有平手 (Tie-break) 情境。
- **動態教學**：根據選擇的遊戲種類，即時顯示相關規則提醒。

## 技術優化
- **強型別支援**：全面使用 TypeScript 定義介面，確保程式碼穩定性。
- **嚴謹計分**：修正了早期版本中踢腳牌判定權重不足的問題。
- **UI/UX 強化**：使用 Tailwind CSS 與 Lucide-react 打造現代化且具備教學引導的介面。

## 本地開發

