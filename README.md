# DealerPro - 專業荷官訓練工具

這是一個專為德州撲克荷官與玩家設計的練習工具，旨在訓練快速核算底池與判斷牌力勝負的能力。

## 核心功能

### 1. 挑戰模式 (Challenge Mode) 🔥
- **限時衝刺**：5 分鐘倒數計時，挑戰在時間壓力下的準確度。
- **排行榜規則 (方案 A)**：僅限挑戰模式產生的分數可登錄排行榜，確保競爭的公平性。
- **累積計分**：挑戰中答錯不重置總分，僅中斷連勝加成，鼓勵在限時內爭取最高產出。

### 2. 計分倍率系統 ⚖️
- **連勝加成 (Streak)**：每連續答對一題，下一題的分數增加 **20%**。保持連勝是衝榜的關鍵。
- **速度獎勵 (Time Bonus)**：10 秒內完成答題享有 **1.5 倍** 加成。
- **難度係數**：
  - **Texas Hold'em**: 1.0x
  - **Omaha**: 1.5x (強制 2+3 規則)
  - **BIGO**: 2.0x (Hi-Lo 高低牌規則)

### 3. 底池分配練習 (Split Pot)
- **多邊池模擬**：自動產生主池 (Main Pot) 與多個邊池 (Side Pot)。
- **餘數處理 (Odd Chips)**：遵循專業規則，將無法整除的零錢優先分配給 ID 較小的贏家。

### 4. 勝負判斷練習 (Showdown)
- **精確判定**：穩定且嚴謹的牌力權重系統，完美處理平手判定。
- **教學引導**：動態顯示各遊戲種類的規則（如奧馬哈強制 2+3、BIGO 的 8-or-better 低牌規則）。

## 本地開發
```bash
npm install
npm run dev
```
